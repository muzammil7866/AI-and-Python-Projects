
# AI A3 - N Queens Search

## Overview

This project packages a small AI search experiment around the 8-queens problem. It compares hill climbing, stochastic hill climbing, and simulated annealing on randomized boards and prints the resulting conflict counts.

## Business Goal

The goal is to demonstrate how local-search techniques behave on a constrained optimization problem. In a portfolio context, this shows experimentation, algorithm comparison, and the ability to turn an academic exercise into a reusable script.

## Structure

- `n_queens_search.py` - core search routines and experiment runner

## Run

```bash
python n_queens_search.py
```

## Notes

This is a teaching and portfolio project rather than a production optimizer.
